### Description
A lightweight and high-performance data logging application for **Rockwell / Allen-Bradley** PLCs (ControlLogix, CompactLogix, etc.). Designed for industrial environments, this tool enables real-time tag monitoring via the Ethernet/IP protocol.

* Robust communication using the **pycomm3** library.
* Simplified data export.
* **Portable Executable**: No Python installation required on the field PC.
